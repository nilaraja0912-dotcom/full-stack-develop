📌 Overview

This system is a scalable full-stack enterprise application built using microservices architecture.

⚙ Setup

# Install dependencies
npm install

# Run server
node server.js

# Start frontend
open index.html

🏗 Architecture

Frontend → API Gateway → Microservices → Database Cluster
        React       Node.js        MongoDB / PostgreSQL

🔗 API Reference

GET /api/users
POST /api/login
GET /api/dashboard
DELETE /api/user/:id

🚀 Deployment

Docker:
docker build -t app .
docker run -p 3000:3000 app

AWS:
- Deploy on EC2
- Use S3 for frontend
- RDS for database
